Detailed Description

A bidirectional in-memory communication channel.

An instance of this class provides a client and server endpoint that are automatically connected to each other similarly to a connected socket.

Test pipes are used to facilitate writing unit tests where the behavior of the transport is tightly controlled to help illuminate all code paths (for code coverage)
